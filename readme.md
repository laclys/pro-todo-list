a simple vue app (support SPA and SSR)

```bash
git clone git@github.com:laclys/pro-todo-list.git
yarn( or npm i)
```

```bash
#build:client
  npm run build:client
```

```bash
#build:server(SSR)
  npm run build:client
  npm run build:server
  # client & server
  npm run build
  # SSR start
  npm start
```

```bash
#run dev:
  npm run dev:client
  npm run dev:server
  # client & server
  npm run dev
```
